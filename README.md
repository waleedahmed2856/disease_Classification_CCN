Skin Disease Classification using CNN

This Convolutional Neural Network (CNN) based image classification model that detects skin conditions from images.
The model is trained to classify images into the following categories:

* Acne and Rosacea Photos
* Bullous Disease Photos

* Image classification using CNN
* Supports RGB images
* Predicts disease category with confidence score
* Easy to integrate with APIs (FastAPI / Flask)
* Lightweight and beginner-friendly implementation

* Python
* TensorFlow / Keras
* NumPy
* PIL (Python Imaging Library)
